# TextFind

TextFind is a platform for execution-time governance in AI systems.

It introduces a model where:

- AI can propose actions  
- systems enforce what is allowed to execute  
- control is applied at the execution boundary

## RFCs

See: `textfind-rfcs/`

## Core Idea

AI systems don’t fail when they generate.  
They fail when they act.

TextFind focuses on controlling execution—not just reasoning.
